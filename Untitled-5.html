Безопасность MAX

О системе безопасности MAX:

MAX использует единую систему безопасности с VK ID. Защита аккаунта MAX напрямую зависит от защиты вашего VK ID — при компрометации VK страдает и MAX.

Ключевые элементы защиты:

  • Единый аккаунт VK ID для авторизации
  • Двухфакторная аутентификация через настройки VK
  • Контроль активных сеансов
  • Управление доступом сторонних приложений

Пошаговая инструкция: Как защитить аккаунт

Шаг 1. Защитите VK ID

Поскольку MAX работает через VK ID, начните с полной защиты VK-аккаунта (см. раздел VK на нашем сайте). Включите 2FA, беспарольный вход или биометрию.

Шаг 2. Проверьте привязки в MAX
  1. Откройте приложение MAX
  2. Профиль → Настройки → Аккаунт
  3. Убедитесь, что привязан правильный VK ID и номер телефона
Шаг 3. Проверьте активные сеансы

В настройках безопасности VK ID просмотрите устройства с доступом к MAX. Завершите сеансы на незнакомых устройствах.

Шаг 4. Настройте приватность профиля
  • Видимость профиля: открытый / только контакты
  • Скрытие номера телефона от незнакомых пользователей
  • Ограничение статуса «в сети»
Шаг 5. Пройдите квиз «Мошенник не пройдёт»

Встроенная функция в MAX для понимания основных схем обмана

Индивидуальный чек-лист приватности:

Пароль VK ID сложный и уникальный
2FA включена в настройках VK
Активные сеансы проверены и очищены
Подозрительные устройства удалены
Сообщения от незнакомцев с ссылками игнорируются и блокируются
Заявки в друзья от незнакомцев не принимаются без проверки
При взломе: сменить пароль VK ID, завершить все сессии, обратиться в поддержку MAX через приложение
